# jRandom
Helper library for easy random object generation.

## Gradle

At your top-level `build.gradle` file:

    repositories {
        // ...
        maven { url 'https://jitpack.io' }
    }
    
And then at your project `build.gradle` file:

    dependencies {
        compile 'com.github.bandyguy:jRandom:0.1'
    }
